# Análise de Vendas

Este projeto realiza uma análise de vendas fictícias, calculando o total de vendas, média de descontos e lucro por produto. Também gera gráficos para visualização dos dados.

## Requisitos

- Python 3.x
- pandas
- matplotlib

   pip install pandas matplotlib
